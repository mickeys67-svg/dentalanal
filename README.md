# 🏥 DentalAnal Marketing Dashboard

> [!IMPORTANT]
> **핵심 아키텍처 안내**: 이 프로젝트는 **Supabase(SQL + JSONB)** 단일 DB 스택을 사용합니다. 수정을 시작하기 전 반드시 [ARCHITECTURE.md](file:///e:/dentalanal/ARCHITECTURE.md)를 숙지하십시오.

## Project Structure
- `backend/`: FastAPI 기반 마케팅 데이터 엔진
- `frontend/`: Next.js 기반 대시보드 UI
- `ARCHITECTURE.md`: 기술 스택 및 데이터 레이어 정의서
- `scripts/`: DB 마이그레이션 및 관리 도구
